## Como usar

#### Pré-requisitos

Ferramentas

- Yarn/Npm

Serviços

- [Ter a api rodando](https://github.com/rodrigodasilva/growthtech-api)

#### Rodando a aplicação

1. Rodamos o comando yarn para fazer a instalação das dependências passadas no package.json

   > yarn

2. Rodamos a aplicação

   > yarn start
